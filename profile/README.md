# SolidWorks Software — Industrial Design CAD Environment (Compatibility Layer)

## About SolidWorks Software

SolidWorks Software provides a complete industrial design CAD environment via compatibility layer (CrossOver/Wine/Parallels) featuring parametric modeling, sheet metal, weldments, advanced assembly motion analysis, full feature tree editing, mold flow simulation, fatigue analysis, cost estimation, and reverse engineering import (mesh to solid). SolidWorks Software combines Design Library with Toolbox hardware (ANSI/DIN/ISO/JIS standard parts), photorealistic rendering (Visualize), and CAM plugin integration directly via emulated configuration. SolidWorks Software enables users to create complex mechanical assemblies, simulate motion, and generate production drawings. SolidWorks Software runs on portable workspaces and dedicated engineering stations (Intel Apple Silicon via Rosetta 2 + compatibility layer). SolidWorks Software is trusted by mechanical engineers, industrial designers, product developers, automotive parts suppliers, and tooling manufacturers worldwide.

## Deep Dive — Parametric Modeling Notes

Feature‑based modeling: sketch extrude (blind, through all, up to surface, up to body, midplane, symmetric, offset from surface), revolve (axis select, 360‑degree, blind angle), sweep (profile along path, guide curve, twist propagation), loft (multiple sections, connectors, start/end tangency, non‑parallel plane sections). Direct editing (move face, offset surface, delete face (patch/fill), combine (common/union/subtract), split (body parting line).
